Enumerable Methods
March 20th, 2015

The enunumable method is used in every major collection class in Ruby. Therefore, it's really important to fully understand to enumerable methods. I am going to break it down for you to help you be more proficient in Ruby moving forward. These instance methods include:

Boolean Queries: returns true or false
  • :all?
  • :any?
  • :include?
  • :none?
  • :one?
    • Examples

    For arrays, you would write the following:

    Ex: Is there one, and only one, state with "West" in its name?

    >> states.one? {|state| state =~ /West/ }

    => true


    Ex: Does the array include Louisiana?

    >> states.include?("Louisiana")

    => true


    For hashes, you would write the following:

    Ex:1.1: Is there one, and only one, state with "West" in its name?

    >> states.keys.one? {|state| state =~ /West/ }

    => true


    OR


    Ex1.2: Is there one, and only one, state with "West" in its name?

    >> states.values.one? {|state| state =~ /West/ }

    => true


    Ex:2.1: Does the array include Louisiana?

    >> states.keys.include?("Louisiana")

    => true


    OR


    Ex:2.2: Does the array include Louisiana?

    >> states.values.include?("Louisiana")

    => true
