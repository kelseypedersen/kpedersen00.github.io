Arrays and Hashes
March 15th, 2015

Arrays and Hashes two separate ways to store information. Arrays are used when you have a list of objects. Hashes are used when you have a list of objects tied to a specific value.

Arrays are an ordered list - A, B, C, D.

To create a new array:

Option #1: new_array = Array.new

Option #2: new_array = []

What an array looks like:

[1, 5, 7]

[apple, banana, kiwi, strawberry]

Hashes are a collection of key/value pairs - Kelsey: Palo Alto, Tom: San Francisco, Sally: San Diego. The keys are: Kelsey, Tom and Sally. The values are Palo Alto, San Francisco and San Diego. Hashes are similar to dictionaries. You can use the key to find the associated value and the value to find the associated key.

To create a new hash:

Option #1: new_hash = Hash.new

Option #2: new_hash = []

What a hash looks like:

[1, 5, 7]

[apple, banana, kiwi, strawberry]
